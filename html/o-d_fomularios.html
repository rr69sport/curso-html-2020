Inputs para fechas

En esta sección hablaremos sobre los inputs que existen para fechas.

Input de tipo date

<input type="date">
Se utiliza para introducir una fecha, tiene un aspecto diferente en cada navegador pero a efectos prácticos en todos hace lo mismo. Los datos que introduce este input es: día, mes y año.

Input de tipo datetime (obsoleto)

Este input ya no se usa, está completamente obsoleto, y si se usase, solo mostraría un input de tipo texto.

Input de tipo datetime-local

<input type="datetime-local">
Se utiliza para introducir fecha y hora. No funciona en firefox.

Un TIP que podría servir para resolver el problema de que no funcione en firefox, es poner un <input type="date"> y un <input type="time">, que sirve para introducir una hora.

Input de tipo month

El <input type="month"> sirve como su nombre indica para introducir un mes del año, como en todos los demás inputs, cambia su aspecto en cada navegador pero el comportamiento es el mismo, permite además elegir un año anterior al actual si es necesario.

Input de tipo week

Lógicamente sirve para elegir una semana de un año en concreto, igual que el de tipo month, puedes elegir un año anterior al actual.

El curso lo dicta Dorian Desings (Canal de YouTube), y tiene la validación de la W3C. El curso se basa en la documentación brindada por la MDN.
