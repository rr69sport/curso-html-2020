Tablas - Parte I

Las tablas en HTML sirven para mostrar contenido tabulado, por ejemplo los horarios de clases de una persona.

Estructura básica

table
Etiqueta que encierra una tabla.
tr
table row, etiqueta que construye una fila.
td
table data, etiqueta que construye una celda.

Importante saber: el número de celdas de un td establecerá el número de columnas que tenga la tabla.

Forma de estructurar una tabla básica

Como puedes ver, la celda con el mismo número de cada fila conforman una columna.
