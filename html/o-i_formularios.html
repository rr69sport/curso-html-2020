Etiqueta datalist

La etiqueta <datalist></datalist> sí es un input y vino con la especificación de HTML5, es muy parecida a la etiqueta select solo que añade un filtro para para que se pueda buscar de una forma más sencilla.

para usarla se declara un input de tipo type="list" y lleva además otro atributo que se llama list="" donde le indicamos la lista de opciones que mostrará al usuario. En el atributo list="" declaramos el id="" que tenga nuestro <datalist id="algo"></datalist>. Para declarar las opciones que tendrá nuestro datalist se anidan etiquetas <option></option> igual que en la etiqueta select

Si eso lo vemos en el navegador, se vería algo así.

Como se puede ver, muestra dos veces el mismo nombre, eso es porque el de arriba es el value="" del option, y el de abajo es el texto dentro de la etiqueta, el cual toma como una descripción del value. Se puede omitir el texto de la etiqueta y dejar solo el value como opción para elegir, o si se necesita se pone una descripción pequeña.

El input que muestra con el cursor, es para que se pueda escribir y filtrar de esa manera las opciones que se muestran.
