Seleccionar columnas

Cuando necesitamos seleccionar una columna, tenemos la etiqueta colgroup, que nos permite seleccionar una columna en concreto. Dentro pondremos tantas etiquetas col como columnas tengamos, cada etiqueta col equivaldrá a una columna siguiendo el mismo orden que tienen en la tabla.

Si necesitamos que una etiqueta col agrupe más de una columna, tenemos el atributo span que funciona exactamente igual que rowspan y colspan.

Como se puede ver, como solo hay una etiqueta col, solo se selecciona la primera columna.

Vamos a suponer que queremos seleccionar la columna 3, como se menciona al comienzo de este artículo, tenemos el atributo span que nos permite seleccionar cuantas columnas necesitemos. Veamos un ejemplo de como usarlo para seleccionar la tercera columna.

Como NO dar estilos a una tabla.

Para estilos existe CSS, por lo tanto no se debe dar estilos directamente con etiquetas html como vimos en la sección elementos de línea, que en el caso de las tablas se puede ver que usen <table border="1" ></table> para darle un borde a la tabla o <center></center> para centrarla. La etiqueta <center> es obsoleta desde finales de la versión 4 de HTML.

El curso lo dicta Dorian Desings (Canal de YouTube), y tiene la validación de la W3C. El curso se basa en la documentación brindada por la MDN.
