Etiqueta select

Select básico

La etiqueta <select></select> crea una lista de opciones donde podemos elegir una o varias opciones. No es un input como tal pero es una mezcla entre los inputs de tipo radio y los de tipo checkbox.

Las opciones desplegadas aparecen cuando se da click en la flechita del recuadro marcado en rojo. El atributo value="" va designado en cada option y el atributo name va designado en la etiqueta select siendo ambos obligatorios porque es el valor que lee el backend cuando se envía el formulario, igual que en los inputs de tipo tipo radio y los de tipo checkbox,.

Por último, para poder seleccionar más de una opción se debe asignar a la etiqueta selecet el atributo booleano multiple, que en ese caso pasa a ser una lista desplegable, lo que muestra es una lista con una barra de scroll.

Para poder seleccionar más de una opción, manteniendo presionada la tecla Ctrl en el teclado, clickeamos sobre las opciones que deseamos elegir.

Select categorías múltiples

La etiqueta select, como vimos, es una lista de opciones que se relacionan entre sí, pero a veces es necesario clasificar en distintos grupos esas opciones. Por ejemplo tenemos una lista de países de distintos continentes, para poder agrupar por categoría las opciones, existe la etiqueta <optgroup></optgroup> y nombrar la categoría con el atributo label="".

El curso lo dicta Dorian Desings (Canal de YouTube), y tiene la validación de la W3C. El curso se basa en la documentación brindada por la MDN.
