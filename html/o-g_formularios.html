Inputs seleccionables

Inputs de tipo radio

<input type="radio" />
Permite seleccionar una única opción de una lista de opciones relacionadas.

Para que este input funcione correctamente, se debe asignar el atributo name="" con el mismo valor en todos los inputs relacionados. Por ejemplo, si queremos saber el sexo del usuario.

Un pequeño tip, es que acepta el atributo booleano checked que lo que hace es que al input de tipo radio que tenga ese atributo, marcarlo por defecto en caso lo necesitemos.

El atributo value="" es obligatorio, ya que es el valor que recibirá junto con el atributo name="" el backend cuando el formuario se envíe.

Inputs de tipo checkbox

<input type="checkbox" />
A diferencia del tipo radio, este input permite seleccionar más de un elemento en una lista de opciones relacionadas.

Por ejemplo si es un formulario para elegir los tipos de lenguajes de programación que quieres aprender.

Como puedes ver, la estructura es muy parecida a la de los inputs de tipo radio. Incluso también acepta el atributo booleano checked por si necesita que una o varias opciones estén seleccionadas por defecto.

Por último, así como está en la imagen el código, el texto aparece antes que el input, si necesita poner el input antes que el texto, simplemente cambia el orden del texto y el input y ya estaría el input antes que el texto.

El curso lo dicta Dorian Desings (Canal de YouTube), y tiene la validación de la W3C. El curso se basa en la documentación brindada por la MDN.
